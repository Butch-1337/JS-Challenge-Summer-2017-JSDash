=============================

### **Instruction to run my version of bot:**

* `git clone` < this repository url >
* `cd ai-jsdash/game`
* `npm i`
* `node jsdash.js --ai=Butch-ai.js --log=log.json --force`

P.S. Write `node jsdash.js --help` to see full list of commands.
